# Personal Portfolio Website - John Aldrie Baquiran

A responsive and modern personal portfolio website designed to showcase web development skills, academic projects, and professional experience. The site features a clean user interface with glassmorphism effects, smooth animations, and a fully functional contact system.

## Features

* **Responsive Design**: Built with Bootstrap 5 to ensure a seamless experience across desktops, tablets, and mobile devices.
* **Interactive UI**: Includes a floating background animation, sticky navigation bar with scroll effects, and scroll-reveal animations for content.
* **Project Showcase**: Dedicated section displaying featured projects (Quick Crave Caf√©, 7AM Coffee POS, etc.) with descriptions and technology badges.
* **Skills & Expertise**: Categorized display of technical skills including Frontend, Backend, UI/UX, and Database Design.
* **Experience Timeline**: A vertical timeline visualizing the educational and professional journey.
* **Functional Contact Form**: Integrated with EmailJS to send messages directly to email, featuring SweetAlert2 for success/error notifications.
* **Glassmorphism Style**: Modern design aesthetic using translucent card backgrounds and subtle borders.

## Tech Stack

* **Core**: HTML5, CSS3, JavaScript
* **Framework**: Bootstrap 5.3.2
* **Icons**: Font Awesome 6.4.0
* **Fonts**: Google Fonts (Inter)
* **Libraries & APIs**:
    * EmailJS (Form submission)
    * SweetAlert2 (Custom popup alerts)
